Screen Match 🎬

Meu primeiro projeto em Java, desenvolvido para praticar os conceitos básicos da linguagem como variáveis, condicionais, loops e entrada de dados pelo usuário.

📚 Sobre o projeto

O projeto simula pequenas funcionalidades relacionadas a filmes, como:

Exibir informações de um filme
Verificar condições com if/else
Ler dados digitados pelo usuário
Trabalhar com estruturas de repetição (for e while)
Calcular médias de avaliações

Os exemplos usam o filme Top Gun: Maverick como referência.

🚀 Tecnologias utilizadas
Java
IntelliJ IDEA
Scanner para entrada de dados
📂 Estrutura do projeto
Main.java

Classe principal do projeto.
Exibe informações do filme e realiza cálculos simples.

Exemplos:

Ano de lançamento
Média de avaliações
Sinopse
Conversão de tipos (casting)
Condicional.java

Prática de estruturas condicionais usando if e else.

Exemplos:

Verificar se o filme é lançamento
Verificar se está incluso no plano
Leitura.java

Prática de leitura de dados com Scanner.

O usuário informa:

Filme favorito
Ano de lançamento
Avaliação
Loop.java

Uso do for para repetir avaliações e calcular média.

OutroLoop.java

Uso do while para permitir avaliações até o usuário digitar -1.

▶️ Como executar
Instale o Java JDK
